### HOW To Use
- 在[your_config.json](./your_config.json)下修改你的信息
- 运行程序
```
npm install
npm run
```
- 查看运行结果
  - 如果控制台上的结果为`10〒/wjx/join/complete.aspx?q=55693469&JoinID=103917325529&jidx=9`，则已经正确提交
  - 如果控制台上的结果为`2〒很抱歉，有异常发生，您刚才填写的答卷没有正常提交，请尝试重新提交或者刷新页面再重新填写`,说明提交错误
